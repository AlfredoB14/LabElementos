Curso Básico de Arduino

Alfredo Barranco Ahued
Ing. Sistemas Computacionales
Fecha: Agosto 23, 2023
Arduino Electrónica

- Resumen -

La práctica consistió en realizar diferentes circuirtos de arduino de diferentes armados y códigos para empezar a familiarizarnos con la sintaxis del lenguaje C y las conexiones desde la placa Arduino hacia las protoboards. Los circuitos se realizaron a partir de un curso básico de Arduino que se nos presentó por la plataforma de Tinkercad, de donde nos guiamos para armar el circuito y copiar el código que se iba a utilizar para el mismo.

- Introducción -

Una placa Arduino es una placa de circuito impreso que contiene un microcontrolador, memoria, puertos de entrada y salida y otros componentes electrónicos. El microcontrolador es el cerebro de la placa y es responsable de ejecutar el código. La memoria almacena el código y otros datos. Los puertos de entrada y salida permiten conectar la placa a otros dispositivos.

Las placas Arduino se pueden programar utilizando el lenguaje de programación Arduino. Arduino es un lenguaje de programación de alto nivel que es fácil de aprender. Hay muchas bibliotecas y ejemplos disponibles para ayudar a los usuarios a comenzar.

Las placas Arduino se pueden utilizar para crear una amplia gama de proyectos, desde simples proyectos de bricolaje hasta complejos sistemas de control. Algunos proyectos comunes incluyen robots, sensores, sistemas de iluminación y dispositivos interactivos.
