Arduino Electrónica Attiny85

- Resumen -

La práctica consistió en realizar diferentes circuirtos con el microcontrolador Attiny85 usando la placa Arduino como programador para correr los códigos.

- Introducción -

Una placa Arduino es una placa de circuito impreso que contiene un microcontrolador, memoria, puertos de entrada y salida y otros componentes electrónicos. El microcontrolador es el cerebro de la placa y es responsable de ejecutar el código. La memoria almacena el código y otros datos. Los puertos de entrada y salida permiten conectar la placa a otros dispositivos.

Las placas Arduino se pueden programar utilizando el lenguaje de programación Arduino. Arduino es un lenguaje de programación de alto nivel que es fácil de aprender. Hay muchas bibliotecas y ejemplos disponibles para ayudar a los usuarios a comenzar.

El Attiny85, un pequeño microcontrolador, con el que podremos realizar pequeños proyectos, en los que no necesitemos gran cantidad de memoria ni de entradas o salidas.

- Desarrollo -

La práctica comenzó con el proceso de convertir la placa Arduino en un programador para poder programar el Attiny85. Este proceso consistió en los siguientes pasos, todo dentro de la IDE de arduino:

1. Importar la librería de Attiny

2. Configurar el Arduino como ISP

3. Configurar el Attiny85

4. Quemar el Bootloader

5. Probar el código de ejemplo

El desarrollo de la práctica consiste en 3 pasos, la simulación, el diseño y la construcción, a continuación se desgloza más de cada uno de estos puntos para ver cómo se llegó a los resultados de la práctica.

Simulación

La simulación de los 13 circuitos se realizó por medio de Tinkercad, donde antes de realizar el armado, se probó cómo funcionaba cada uno para poder verificar si estaba funcionando correctamente o no. Después de este proceso, al saber que cómo funcionaban los circuitos se armaba cada uno en orden después de su simulación.

Construcción

La construcción de los 13 circuitos se basó en usar los materiales mencionados al principio del reporte para construir cada uno de los ejemplos que se nos proporcionó por el curso básico de Arduino en Tinkercad.

Fue un proceso en el que se tiene que prestar muchísima atención, ya que una mala construcción puede llevar a un corto circuito de alguno de los componentes.

- Resultados -

A continuación se prensentan los resultados, videos de cómo sirvió cada uno de los 13 circuitos armandos con el Arduino y el Attiny, así como una pequeña explicación de que hace cada uno de ellos.

Reportes de Prácticas:

Circuito 1: Attiny Blink

Se hace parpadear un LED.

Circuito 2: Attiny Digital HIGH

Usar la salida Digital HIGH para dejar prendido el LED.

Circuito 3: Attiny Digital LOW

Usar la salida Digital LOW para dejar APAGADO el LED.

Circuito 4: Attiny Digital Delay

Con el uso de un delay y las salidas digitales HIGH y LOW se realiza un parpadeo a un ritmo más lento dependiendo del valor del delay.

Circuito 5: Attiny 2 LEDs Delay

Con HIGH, LOW y un Delay se hace parpadear de un LED a otro.

Circuito 6: Attiny 3 LEDs Delay

Con HIGH, LOW y un Delay se hace parpadear de un LED a otro.

Circuito 7: Attiny Digital Input

Se usa el Input para conseguir la señal del botón que prenderá el LED.

Circuito 8: Attiny Digital Input IF-ELSE

Se hace el código de tal manera que se usa una condicinal IF y ELSE del input del botón para el prendido y apagado del LED.

Circuito 9: Attiny DIgital Input IF variables

Se hace el código de tal manera que se usa una condicinal IF de una variabe que contiene el input del botón para el prendido y apagado del LED.

Circuito 10: Attiny Digital Input OR

El LED se prende siempre y cuando se cumpla la condicional OR con los dos botones.

Circuito 11: Attiny Digital Input AND

El LED se prende siempre y cuando se cumpla la condicional AND con los dos botones.

Circuito 12: Attiny Contador + 3 LEDs

Consiste en 3 LEDs los cuales se irán prendiendo uno a uno con el botón.

Circuito 13: Attiny Contador + - 3 LEDs

Consiste en 3 LEDS y dos botones, un botón sive para ir prendiendo los LEDs uno a uno, mientras el otro botón servirá para ir apagando de uno en uno.

- Conclusiones -

Esta práctica nos ayudó a aprender cómo configurar una placa Arduino para que sirva como un programador para el Attiny, así como más práctica haciendo los circuitos electrónicos y la diferencia entre hacerlo directo desde el Arduino y hacerlo con el Attiny.
